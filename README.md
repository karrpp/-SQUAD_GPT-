# SQUAD GPT — 성민 FC

> 1인 스튜디오 + AI 에이전트 팀으로 운영되는 프로젝트

**"다양한 AI 자아와 협업하며, 사람들에게 진짜 도움이 되는 프로덕트를 만든다."**

---

## 프로젝트 소개

성민 FC는 축구 구단 운영 방식을 차용한 AI 에이전트 팀입니다.
감독(성민)이 주제를 던지면, 6명의 핵심 에이전트가 각자의 포지션에서 협업하여 프로덕트를 완성합니다.

이 리포지토리는 성민 FC의 **공식 홈페이지**입니다.

---

## 핵심 6인 에이전트

| 에이전트 | 포지션 | 역할 |
|----------|--------|------|
| **Guardiola** | 감독 | 전략 종합, 실행 플랜 수립 |
| **Messi** | 캡틴 | 방향 설정, 위기 중재, 팀 리딩 |
| **CR7** | ST | 임팩트 아이디어 + 끝까지 실행 |
| **Modric** | CM | 아이디어 → 실행 가능한 태스크 변환 |
| **Buffon** | GK | 배포 전 최종 검증, 품질 보증 |
| **Pedri** | 유틸리티 MF | 빈틈을 메우는 루틴 업무 전담 |

---

## 핵심 가치

| 가치 | 설명 |
|------|------|
| **간결함** | 복잡한 것을 단순하게. 불필요한 것을 덜어내라. |
| **완성도** | 타협 없이 끝까지. 중간에 멈추지 마라. |
| **헌신** | 빈틈을 메우는 묵묵한 실행. 화려하지 않아도 된다. |
| **기록** | 모든 과정을 남긴다. 왜 그렇게 했는지를 적어라. |

## 기술 스택

- **React 19** + **Vite 7** (SPA)
- **react-router-dom** — 클라이언트 사이드 라우팅
- **CSS Variables** — 디자인 토큰 시스템
- **ESLint 9**, ES Modules
- **Vercel** 배포 예정

## 프로젝트 구조

```
SQUAD_GPT/
├── src/
│   ├── pages/          ← 페이지 컴포넌트
│   ├── sections/       ← 홈페이지 섹션 (Hero, About, Values, Squad, ...)
│   ├── components/     ← 공통 컴포넌트 (Navbar, PlayerCard, ...)
│   ├── styles/         ← 디자인 토큰 (tokens.css)
│   └── data/           ← 에이전트 데이터
├── playbook/
│   ├── prompts/        ← 에이전트 프롬프트 (core + reserves)
│   ├── standards/      ← 코딩/아키텍처/토큰 예산 규칙
│   └── templates/      ← 킥오프/회고/스크럼 템플릿
├── .claude/agents/     ← Claude Code 에이전트 설정
└── CLAUDE.md           ← 팀 운영 규칙
```

## 경기 운영법

```
1️⃣ 킥오프      → 감독이 주제를 던진다
2️⃣ 캡틴 리딩   → Messi가 방향을 잡는다
3️⃣ 전술 수립   → Guardiola가 전략을 짠다
4️⃣ 공격 전개   → CR7이 임팩트 아이디어를 실행한다
5️⃣ 태스크 변환  → Modric이 구체적 태스크로 분해한다
6️⃣ 최종 세이브  → Buffon이 품질을 검증한다
7️⃣ 골!         → 감독이 최종 승인한다
```
