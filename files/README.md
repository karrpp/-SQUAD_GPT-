# ⚽ 성민 FC

> AI 에이전트 스쿼드와 함께 프로덕트를 만드는 1인 스튜디오

---

## 미션

**"다양한 AI 자아와 협업하며, 사람들에게 진짜 도움이 되는 프로덕트를 만든다."**

## 비전

- 축구팀처럼 구성된 AI 에이전트들이 각자의 역할을 수행하며
- 매니저(성민)의 아이디어를 실제 작동하는 프로덕트로 만들어내는
- 체계적이고 재현 가능한 개발 시스템을 구축한다

## 핵심 가치

1. **간결함 (Simplicity)** — 복잡한 것을 단순하게. Griezmann처럼.
2. **완성도 (Quality)** — 타협 없이 끝까지. CR7처럼.
3. **헌신 (Dedication)** — 빈틈을 메우는 묵묵한 실행. Park처럼.
4. **기록 (Documentation)** — 모든 과정을 남긴다. Zidane처럼.
