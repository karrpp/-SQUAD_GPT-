# inbox/

> 폰(Termux/리눅스 모바일 환경)에서 작성한 파일의 임시 착지점.

## 용도

- 이동 중 폰으로 작성한 메모, 아이디어, 회고
- 정식 폴더로 분류되기 전 파일을 여기 두고 나중에 정리
- **성민이 주기적으로 직접 삭제**

## 규칙

- 파일명: 자유 (폰에서 빠르게 작성한 그대로)
- 정식 폴더에 복제된 파일은 여기서 삭제해도 됨
- 장기간 쌓이면 Pedri에게 정리 요청

## 정식 분류 기준

| 내용 유형 | 이동 경로 |
|-----------|-----------|
| 에이전트 토의 | `playbook/discussions/` |
| 개인 회고/경험 | `playbook/discussions/` + 옵시디언 `retrospectives/` |
| 아이디어/킥오프 | `playbook/discussions/` |
| 코드 메모 | 해당 `src/` 또는 `playbook/standards/` |
