# discussions/

> 에이전트 간 토의록 및 전략 논의 기록 보관소

## 목적

- 주요 의사결정 과정 기록
- 에이전트 간 협업 결과물 보관
- 전략 수립 및 방향 설정 문서 아카이브

## 파일명 규칙

```
YYYY-MM-DD_에이전트1-에이전트2_주제.md
```

**예시:**
- `2026-02-12_Guardiola-Messi_노트앱-피벗-전략.md`
- `2026-02-15_CR7-Modric_MVP-개발-계획.md`

## 문서 구조 (권장)

1. **상황 정리** — 배경, 요청 사항
2. **에이전트별 의견** — 각자의 관점과 분석
3. **토의 내용** — 주요 쟁점과 논의
4. **최종 합의** — 결론 및 액션 아이템
5. **다음 단계** — 후속 작업 및 담당자

## 보관 정책

- 모든 주요 토의는 이 폴더에 저장
- 3개월 이상 지난 문서는 `discussions/archive/` 이동 (선택)
- 삭제하지 않고 아카이브로 유지 (기록 중시)
