## 사용법

### Messi가 직접 작성할 때 (opus)

세션 끝에 아래 중 하나로 요청:
```
@Messi 오늘 데일리 스크럼 정리해줘
@Messi 오늘 뭐 했는지 기록해줘
```

### Pedri에게 위임할 때 (haiku, 비용 절약)

템플릿이 안정화된 후:
```
@Pedri 오늘 데일리 스크럼 작성해줘
```
Pedri는 대화 내역을 읽고 템플릿에 맞춰 자동 채움.

## 작성 규칙

1. **WHY 우선**: "뭘 했다"보다 "왜 그렇게 했다"가 중요
2. **고민 기록**: 선택지가 여러 개였을 때, 왜 그걸 골랐는지 남겨야 나중에 맥락 파악 가능
3. **변경 파일 명시**: 나중에 추적할 수 있도록
4. **팀 상태**: 어떤 에이전트가 투입됐는지 기록
5. **간결하게**: 길게 쓰지 말고, 핵심만. Pedri가 인수할 수 있을 정도로 구조화
