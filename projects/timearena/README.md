## 기본 정보

- **코드 레포**: https://github.com/karrpp/timearena-
- **스택**: Expo SDK 54 (React Native + TypeScript)
- **시작일**: 2026-03-06
- **Phase 1**: 6주 — Expo Web 배포
- **Phase 2**: 12주 — iOS/Android
